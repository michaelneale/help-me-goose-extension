Goose App Not Found

It looks like the Goose application is not installed or not properly configured on your system.

To fix this issue:

  1. Make sure the Goose application is installed on your computer.
  2. If already installed, the application may need to be registered to handle goose:// protocol links.
  3. Try restarting your browser after installing or updating Goose.

Browser-Specific Instructions

Chrome
Brave
Edge
Firefox
Safari

Chrome Protocol Handler Setup

  1. Type chrome://settings/handlers in your address bar
  2. Make sure "Allow sites to ask to become default handlers for protocols" is enabled
  3. If Goose is listed under "Site can ask to handle protocols", make sure it's allowed
  4. Restart Chrome and try again

Brave Protocol Handler Setup

  1. Type brave://settings/handlers in your address bar
  2. Make sure "Sites can ask to handle protocols" is enabled
  3. If Goose is listed under "Site can ask to handle protocols", make sure it's allowed
  4. Restart Brave and try again

Edge Protocol Handler Setup

  1. Type edge://settings/content/protocolActivation in your address bar
  2. Make sure protocol activation is properly configured
  3. Restart Edge and try again

Firefox Protocol Handler Setup

  1. When you first click a goose:// link, Firefox should ask how to handle it
  2. If you previously declined, go to Firefox Settings > Applications
  3. Find "goose" in the list and set it to use the Goose application
  4. If it's not listed, you may need to reinstall the Goose application

Safari Protocol Handler Setup

  1. Safari should ask how to handle the goose:// protocol when first encountered
  2. If you previously declined, you may need to reset Safari's protocol handler settings
  3. Alternatively, reinstall the Goose application
